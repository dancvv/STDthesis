\section{构建文档}
XeCJK 是提供 LaTeX 中文支持的宏包，并且依赖于 XeLaTeX，因此，我们需要使用 XeLaTex 命令进行构建。
LaTeX 在构建交叉索引时需要多次运行，才能最终解析所有的引用，并且期间需要 BibTeX 对参考文献数据库进行处理。因此，一般的手动构建命令是：

1.xelatex main

2.bibtex main

3.xelatex main

4.xelatex main
